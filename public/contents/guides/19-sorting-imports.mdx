Сортировка импортов убирает хаос в начале файла.
Когда порядок единый, быстрее читать модуль и меньше конфликтов в merge.
Это также ускоряет навигацию: разработчик сразу понимает, что внешнее, что локальное, а что относится к стилям.

```tsx
import { Button } from './Button';
import React from 'react';
import styles from './Button.module.css';
import clsx from 'clsx';
import { ReactNode, useEffect, useState } from 'react';
```

```tsx
import type { ReactNode } from 'react';
import { useEffect, useState } from 'react';

import clsx from 'clsx';

import { Button } from './Button';

import styles from './Button.module.css';
```

Настройте автосортировку импортов в проекте один раз — дальше порядок будет поддерживаться при сохранении файла, без ручной правки.
